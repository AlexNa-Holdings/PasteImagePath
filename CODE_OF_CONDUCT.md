# Code of Conduct

## Our Commitment

We want this project to be a respectful, harassment-free experience for everyone.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent.
- Focus on technical issues and collaboration.

## Unacceptable Behavior

- Harassment, discrimination, or personal attacks.
- Trolling, insults, or inflammatory comments.
- Publishing private information without permission.

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, issues, and PRs that violate this code.

## Scope

This applies to project spaces including issues, pull requests, discussions, and other community channels.
